qbits.alia.codec.nippy

Codec that adds encoding support for nippy serialized data. Both encoding/decoding are optional. You need to add a nippy dependency manually. It only works for prepared statements (write time)

set-nippy-serializable-encoder!

(set-nippy-serializable-encoder!)(set-nippy-serializable-encoder! opts)

Sets a nippy decoder for all bytebuffers returned by cassandra. Encoding is manual, and only applied to values that are marked by calling serializable! on them. Takes a map or nippy options to be passed to thaw/freeze.
